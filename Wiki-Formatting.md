## Bullet Points:

### Use "* " to create a bullet point (note the space after)
```
* This is a bullet point!
```
* This is a bullet point!

### Use " * " to create a secondary point (note the space before)
```
* Primary bullet!
  * This is a secondary bullet point!
```
* Primary bullet!
  * This is a secondary bullet point!

### External link
```

```

### Internal Link (actual)
```

```

```

```
